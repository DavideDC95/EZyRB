Module to handle STL files.

Vtk format file handler class. You are NOT supposed to call directly this class constructor (use FileHandler constructor instead)

Parameters:

filename (str) – name of file
Variables:
  • _filename (str) – name of file to handle
  • _cached_data (vtkPolyData) – private attribute to store the last polydata processed
_read_polydata()[source]

This private method reads the given filename and return a vtkPolyData object containing all informations about file; to avoid useless IO operation on the same file, it stores polydata of the last file parsed and if user ask for this file, the polydata previously stored is returned.

Returns:polydata containing information about file.
Return type:vtkPolyData
_save_polydata(data, write_bin=False)[source]

This private method saves into filename the data. data is a vtkPolydata. It is possible to specify format for filename: if write_bin is True, file is written in binary format, otherwise in ASCII format. This method save cached polydata to reduce number of IO operations.

Parameters:
  • data (vtkPolyData) – polydatat to save.
  • write_bin (bool) – for binary format file.
get_geometry(get_cells=False)[source]

This method reads the given filename and returns points and cells of file. If get_cells is True, it computes the list that contain index of points defining cells, otherwise the list is not computed and set to None (less expensive).

Parameters:get_cells (bool) – flag to compute cells list or not. Default is false.
Returns:the n_points-by-3 matrix containing the coordinates of the points, the n_cells list containing, for each cell, the index of the points that define the cell (if computed).
Return type:numpy.ndarray, list(numpy.ndarray) or None
set_geometry(points, cells, write_bin=False)[source]

This method save into filename a new data defined by points and cells.

Parameters:
  • points (numpy.ndarray) – matrix n_points-by-3 containing coordinates of all points.
  • cell (list(list(int))) – a list of lists that contains, for each cell, the index of points defining the cell.
  • write_bin (bool) – flag to write in the binary format. Default is false.
