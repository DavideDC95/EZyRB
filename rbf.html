class RBFInterpolator(points, values, radius=1.0, norm='euclidean', basis=None)

Bases: object

Multidimensional interpolator using Radial Basis Function.

Parameters:
  • points (array_like) – the coordinates of the points.
  • values (array_like) – the values in the points.
  • radius (float) – the radius used in the basis functions. Default is 1.0.
  • norm (str or callable) – The function that returns the distance between two points. It has to be a function that take as input a vector and return a float number. Default is ‘euclidean’.
  • basis (callable) – the basis function.
Variables:
  • points (array_like) – the coordinates of the points.
  • radius (float) – the radius used in the basis functions. Default is 1.0.
  • basis (callable) – the basis function.
  • weights (numpy.ndarray) – the weights matrix.
