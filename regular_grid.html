class RegularGrid[source]

Bases: ezyrb.approximation.Approximation

Multidimensional interpolator on regular grids. See also scipy’s RegularGridInterpolator for information on kwargs.

Parameters

  • points (array_like) – the coordinates of the points on a regular grid.

  • values (array_like) – The (vector-) data on the regular grid in n dimensions.

Example
>>> import ezyrb
>>> import numpy as np
>>> def f(x, y, z):
...     return 2 * x**3 + 3 * y**2 - z
>>> x = np.linspace(1, 4, 11)
>>> y = np.linspace(4, 7, 22)
>>> z = np.linspace(7, 9, 33)
>>> xg, yg, zg = np.meshgrid(x, y, z, indexing='ij')
>>> points = np.c_[xg.ravel(), yg.ravel(), zg.ravel()]
>>> data_mode_x = f(xg, yg, zg).reshape(-1, 1)
# lets assume we have 2 modes, i.e. a rank 2 model
>>> data = np.concatenate((data_mode_x, data_mode_x/10), axis=1)
>>> rgi = ezyrb.RegularGrid()
>>> rgi.fit(points, data, method="linear")
>>> pts = np.array([[2.1, 6.2, 8.3],
...                 [3.3, 5.2, 7.1],
...                 [1., 4., 7.],
...                 [4., 7., 9.]])
>>> rgi.predict(pts)
array([[125.80469388,  12.58046939],
       [146.30069388,  14.63006939],
       [ 43.        ,   4.3       ],
       [266.        ,  26.6       ]])
>>> f(pts[:, 0], pts[:, 1], pts[:, 2])
array([125.542, 145.894,  43.   , 266.   ])
>>> f(pts[:, 0], pts[:, 1], pts[:, 2])/10
array([12.5542, 14.5894,  4.3   , 26.6   ])
