ezyrb.vtkhandler.VtkHandler._read_polydata¶

VtkHandler._read_polydata()¶

This private method reads the given filename and return a vtkPolyData object containing all informations about file; to avoid useless IO operation on the same file, it stores polydata of the last file parsed and if user ask for this file, the polydata previously stored is returned.

Returns:polydata containing information about file.
Return type:vtkPolyData
