ezyrb.mapper.Mapper.map_solution¶

Mapper.map_solution(output_file, solution_file, geometry_file=None)¶

This method maps the solution on a new geometry. To be more specific, for each point (or cell) of the given geometry looks for the nearest solution points (or cells) and interpolate the selected output using neighbors values.It is possible select number of neighbors and the interpolate function. If geometry_file is not specified, to-interpolate points are taken from output_file, so in this case output_file must contain geometry information; if geometry_file is specified, to-interpolate points are taken from geometry_file and the interpolated solution is write on output_file.

Parameters:
  • output_file (str) – name of file where interpolated solution will be stored. Only vtk file are supported.
  • solution_file (str) – name of file where real solution is stored.
  • geometry_file (str) – name of file where geometry information is stored.
