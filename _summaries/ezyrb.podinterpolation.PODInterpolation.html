generate(points, snapshots[, interpolator]) Generate the reduced space using the proper orthogonal decomposition interpolation: the matrix that contains the snapshots computed for the points is decomposed using SVD tecnique to obtain the POD basis.
load() Load the space from a file.
loo_error(snapshots[, func]) Compute the error for each parametric point as projection of the snapshot onto the POD basis with a leave-one-out (loo) strategy.
save(filename) Save the space on file.
