static default_interpolate_func(distance)

Default function for the point interpolation. Take as argument the neighbors values and the respective distance between to-interpolate point and neighbors, and return the values weighted average (weights are the reciprocal of the distances). If to-interpolate point coincides with one of the neighbors, return that neighbour value.

map_solution(output_file, solution_file, geometry_file=None)

This method maps the solution on a new geometry. To be more specific, for each point (or cell) of the given geometry looks for the nearest solution points (or cells) and interpolate the selected output using neighbors values.It is possible select number of neighbors and the interpolate function. If geometry_file is not specified, to-interpolate points are taken from output_file, so in this case output_file must contain geometry information; if geometry_file is specified, to-interpolate points are taken from geometry_file and the interpolated solution is write on output_file.

Parameters:
  • output_file (str) â name of file where interpolated solution will be stored. Only vtk file are supported.
  • solution_file (str) â name of file where real solution is stored.
  • geometry_file (str) â name of file where geometry information is stored.
