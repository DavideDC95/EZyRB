MatlabHandler¶

Module to handle Matlab files.

MatlabHandler Matlab format file handler class.
MatlabHandler.get_dataset Method to parse the filename.
MatlabHandler.set_dataset Writes to filename the given output.
class MatlabHandler(filename)

Bases: object

Matlab format file handler class. You are NOT supposed to call directly this class constructor (use FileHandler constructor instead).

Variables:
  • _filename (str) – name of file to handle.
  • _cached_data (dict) – private attribute to save a copy of last data processed; it allows to reduce IO operation on the same file. Initial value is None.
get_dataset(output_name)

Method to parse the filename. It returns a matrix with all the values of the chosen output.

Parameters:output_name (str) – name of the output of interest inside the mat file.
Returns:a n_points-by-n_components matrix containing the values of the chosen output.
Return type:numpy.ndarray
set_dataset(output_values, output_name)

Writes to filename the given output. output_values is a matrix that contains the new values of output to write, output_name is a string that indicates name of output to write.

Parameters:
  • output_values (numpy.ndarray) – it is a n_points-by-n_components matrix containing the output values.
  • output_name (str) – name of the output.
